RedStore E-Commerce Website

Project Overview

RedStore is a fully responsive e-commerce website designed for sportswear and lifestyle products. This project implements modern UI/UX principles with a focus on mobile-first design, seamless navigation, and engaging product showcase sections. It serves as a complete front-end demonstration of an online retail platform.

Technologies Used

  • HTML5 (Semantic Elements: header, section, footer)
  • CSS3 (Responsive Layout, Flexbox, Custom Styling)
  • Font Awesome 6.0 (Icons for Rating, Social, Navigation)
  • Vanilla JavaScript (Mobile Menu Toggle Functionality)

Key Features

  • Responsive mobile menu with toggle functionality
  • Section-based layout (Home, Categories, Featured/Latest Products, Offers)
  • Product rating system with star icons
  • Testimonials section with customer reviews
  • Brand showcase and fully responsive footer with social links
  • Smooth scrolling between page sections
